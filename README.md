# MedRAG

## Overview
MedRAG is an AI-powered healthcare research assistant that helps users explore and analyze medical research papers using Retrieval-Augmented Generation (RAG). It provides context-aware answers and summaries from scientific literature.

## Features
- Semantic search of healthcare research papers
- AI-powered question answering
- Research paper summarization
- Knowledge extraction
- Interactive Streamlit dashboard

## Technologies Used
- Python
- Streamlit
- FAISS
- BM25
- Large Language Models (LLMs)

## Project Structure
- agents/
- core/
- data/
- ui/
- app.py
- requirements.txt

## How to Run
1. Install dependencies:
   ```
   pip install -r requirements.txt
   ```
2. Run the application:
   ```
   streamlit run app.py
   ```
